# Target Audience Analysis: Tone Generator

## Overview

This document profiles the primary user segments for our tone generator, their needs, pain points, and how we can serve them better than competitors.

---

## Primary Audience Segments

### 1. Musicians Tuning Instruments

**Demographics**:
- Age: 15-65
- Mix of hobbyists and professionals
- All skill levels
- Global audience

**Instruments**:
- Guitar (largest segment): acoustic, electric, bass
- Orchestral strings: violin, viola, cello, bass
- Other fretted: ukulele, mandolin, banjo
- Wind instruments (reference pitch)
- Piano technicians (specialized)

**Use Cases**:
- Quick reference pitch for tuning by ear
- Verifying electronic tuner accuracy
- Learning to tune by ear
- Alternative tunings (Drop D, Open G, etc.)
- Setting intonation
- Checking harmonics

**Pain Points**:
- Need specific frequencies quickly
- Want multiple string presets
- Alternative tuning references are hard to find
- Need to hear sustained tones while tuning

**What They Value**:
- Instant playback
- Accurate frequencies
- Comprehensive presets
- Clean, sustained tones
- Mobile-friendly

**How to Win Them**:
- Extensive instrument presets
- Alternative tuning library
- Quick-access buttons
- Octave switching
- Integration with our metronome

---

### 2. Audio Engineers & Technicians

**Demographics**:
- Age: 20-55
- Professionals and serious hobbyists
- Technical knowledge of audio
- Studio, live sound, broadcast

**Use Cases**:
- Speaker/headphone testing
- Room acoustic analysis
- Equipment calibration
- Frequency response testing
- Identifying resonances
- Signal chain verification
- Line level testing (1kHz reference)

**Pain Points**:
- Need precise, calibrated tones
- Want frequency sweeps
- Need multiple test frequencies quickly
- Download capability for offline use

**What They Value**:
- Accuracy and precision
- Professional features (sweep, noise)
- Download/export options
- Technical specifications
- No audio artifacts

**How to Win Them**:
- Frequency sweep mode
- WAV export
- Professional test frequency presets
- Noise generation (white, pink, brown)
- Clear technical documentation
- dB SPL guidelines

---

### 3. People Concerned About Hearing

**Sub-segments**:

#### A. Tinnitus Sufferers
- Want to identify tinnitus frequency
- Seeking masking sounds
- Exploring habituation therapy
- Age: typically 40+

#### B. Hearing Test Seekers
- Curious about hearing ability
- Checking for age-related loss
- Monitoring hearing over time
- All ages, especially 40+

#### C. Hearing Protection Conscious
- Musicians protecting hearing
- Industrial workers
- Audiophiles
- Want to understand safe levels

**Pain Points**:
- Uncertainty about tinnitus frequency
- No access to audiologist
- Want private, free assessment
- Concerned about hearing loss

**What They Value**:
- Simple, clear interface
- Privacy (no data collection)
- Educational content
- Safe listening warnings
- Guidance and next steps

**How to Win Them**:
- Simple hearing test mode
- Tinnitus frequency matching
- Audiogram-style results
- Educational content on hearing health
- Clear disclaimers (not medical advice)
- Links to professional resources

**How to Win Them**:

- Academic citations

## User Journey Maps

### Musician Journey
```
1. Need to tune guitar
2. Search "guitar tuning tones" or "440 Hz tone"
3. Land on tone generator
4. Find guitar presets
5. Play each string frequency
6. Tune instrument
7. (Optionally) Bookmark for later
```

**Optimization**: Make guitar presets prominent, fast loading

### Audio Engineer Journey
```
1. Testing new speakers
2. Search "frequency sweep online" or "test tones"
3. Land on tone generator
4. Need sweep function
5. Run frequency sweep
6. Note any issues
7. (Optionally) Download WAV for repeat testing
```

**Optimization**: Add sweep mode, download capability

### Hearing Test Journey
```
1. Worried about hearing
2. Search "online hearing test" or "can I hear 15kHz"
3. Land on tone generator
4. Try high frequency tones
5. Want structured test
6. Take hearing test
7. View results
8. (Optionally) Seek professional help
```

**Optimization**: Add hearing test mode, clear disclaimers

### Student Journey
```
1. Studying acoustics/physics
2. Search "sine wave vs square wave"
3. Land on tone generator
4. Read educational content
5. Experiment with waveforms
6. Try dual oscillator for beats
7. Understand concepts
8. Return for reference
```

**Optimization**: Enhanced educational content, interactive demos

## Accessibility Considerations

1. **Keyboard navigation**: Full control without mouse
2. **Screen reader support**: ARIA labels for all controls
3. **Color contrast**: High contrast dark theme
4. **Touch targets**: Large buttons for mobile
5. **Hearing impaired**: Visual feedback for audio status
6. **Cognitive**: Simple mode for basic use

---

## Geographic Considerations

- **Primary markets**: US, UK, Canada, Australia, Germany
- **Concert pitch**: A=440 Hz standard (mention European 442 Hz)
- **Language**: English only (for now)
- **Mobile usage**: High in developing markets
- **Time zones**: 24/7 availability important
